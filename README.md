<h1>Tweet Generator AI Chatbot</h1>

<p>Welcome to the <strong>Tweet Generator AI Chatbot</strong>, a simple yet powerful tool that helps you create engaging tweets effortlessly. Powered by Google’s Gemini AI model and built with Python and interactive Jupyter widgets, this project makes writing tweets fun and fast—no more staring at blank screens!</p>

<h2>What it does</h2>

<ul>
  <li>Enter a <strong>topic</strong> you want to tweet about</li>
  <li>Choose a <strong>tone</strong> like professional, casual, motivational, or informative</li>
  <li>Specify your <strong>target audience</strong> for a tailored message</li>
  <li>Add relevant <strong>hashtags</strong> to boost your tweet’s reach</li>
</ul>

<p>Once you click <strong>Generate Tweet</strong>, the AI instantly crafts a tweet for you, keeping it short and sharp under Twitter’s 280-character limit.</p>

<h2>Why use it?</h2>

<p>Writing great tweets can be tricky and time-consuming. This tool leverages the power of Google’s Gemini model to take your ideas and turn them into polished social posts in seconds. It’s perfect for content creators, social media managers, marketers, or anyone looking for inspiration.</p>

<h2>How to use</h2>

<ol>
  <li>Run the notebook (or script) in a Jupyter environment like Jupyter Notebook or Google Colab.</li>
  <li>When prompted, enter your <strong>Gemini API key</strong>.</li>
  <li>Fill in the input fields for topic, tone, audience, and hashtags.</li>
  <li>Hit <strong>Generate Tweet</strong> and watch the magic happen!</li>
  <li>The generated tweet will display right below the button.</li>
</ol>

<h2>Requirements</h2>

<ul>
  <li>Python 3.x</li>
  <li><code>google-generativeai</code> package (Google Gemini API client)</li>
  <li><code>ipywidgets</code> for the interactive UI</li>
  <li>A valid <strong>Gemini API key</strong> (get it from Google’s generative AI platform)</li>
</ul>

<p>You can install required packages with:</p>

<pre><code>pip install google-generativeai ipywidgets
</code></pre>
